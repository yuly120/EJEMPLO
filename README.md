
Este repositorio contiene un ejemplo práctico y ejercicios desarrollados con **React**, enfocados en aprender los fundamentos 
 al igual el consumo de apis de fake estore api.
## utiizamos reak para poder diseñar las interfaces al igual que css y js
 respecto a fake estore api se agrego
 1.-un inicio de sesión  con los usuarios regstrados
 2.-se agrego una lista con los usuarios y sus datos
 3.-un apartado de compras con los productos
 4.-un apartado de carrito con varios productos guardados
 5.-un apartado de comida donde puedes verla con sus datos
 6.-tambien muestra el clima
 7.-de igual manera se manejan mapas en tiempo real y utras ubicaciones
##  Notas
Este proyecto es parte de un proceso de aprendizaje sobre desarrollo Frontend.

## Instalación y Uso

1. Clona el repositorio:
   https://github.com/ROB-ARK/ejemplos-reak.git

2. Entra a la carpeta del proyecto:
   cd nombre-app

3. Instala las dependencias:
   npm install

4. Inicia el servidor de desarrollo:
   npm run dev
